\# 🍽️ Restaurant Recommendation System



A content-based restaurant recommendation system built with Python and Flask.



\## 📌 About

This system recommends restaurants based on user reviews using 

TF-IDF vectorization and Cosine Similarity on the 

Zomato Bangalore Restaurants dataset.



\## 🛠️ Tech Stack

\- Python, Pandas, NumPy

\- Scikit-learn, NLTK

\- Matplotlib, Seaborn

\- Flask, HTML, CSS



\## 🚀 How to Run

1\. Install requirements: `pip install pandas numpy matplotlib seaborn scikit-learn nltk flask plotly`

2\. Run Jupyter Notebook in `Model/` folder

3\. Navigate to `Flask/` folder

4\. Run `python app1.py`

5\. Open `http://127.0.0.1:5000`



\## 📊 Dataset

Zomato Bangalore Restaurants — \[Kaggle](https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants)
